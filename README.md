# Pizazoo

Pizazoo is a private Mac screenshot organizer that turns scattered screenshot files into a searchable, structured library.

It watches your screenshot folder, reads captures on-device, suggests useful categories, and helps you find screenshots by the text inside them. No cloud upload. No account required. Your screenshots stay on your Mac.

## What Pizazoo Does

Pizazoo is built for people who take screenshots constantly: developers, designers, founders, students, support teams, researchers, and anyone whose Desktop quietly fills with files named `Screenshot 2026-...`.

Instead of leaving screenshots as loose files, Pizazoo gives them a home:

- A native macOS library for browsing and reviewing captures
- On-device text recognition so screenshots become searchable
- Automatic categories for receipts, code, design, chats, errors, payments, travel, shopping, media, documents, settings, articles, and more
- Custom categories that can learn from how you organize
- A quick processing popup after each screenshot, so you can confirm category, custom category, tags, notes, favorite status, or title while the context is fresh
- Fast search by text, tags, categories, notes, filenames, and captured content
- Duplicate detection and desktop cleanup workflows
- Menu bar access for quick review without breaking focus
- Saved searches, multi-select organization, drag-to-category, favorites, archive, trash, and undo flows
- Local-first processing designed for privacy-conscious Mac users

## Privacy First

Pizazoo is designed around local processing.

Your screenshots can contain invoices, private messages, code, customer data, product plans, credentials, or work-in-progress designs. Pizazoo treats that content carefully:

- Text recognition happens on-device
- Screenshot organization happens locally
- No screenshot is uploaded for classification
- No account is required for the core workflow
- Your library remains under your control on your Mac

## Before Opening an Issue

Please search existing issues first. Someone may have already reported the same behavior.

When reporting a bug, include as much detail as you can:

- Pizazoo version
- macOS version
- Mac model, if relevant
- What you expected to happen
- What actually happened
- Steps to reproduce the issue
- Screenshots or screen recordings, if safe to share
- Whether the issue happens every time or only sometimes

Please do not attach private screenshots, license keys, secrets, passwords, customer data, or confidential work material.

## Good Bug Report Example

Title: `Processing popup clears my custom category after changing the main category`

Helpful details:

- macOS 15.4
- Pizazoo 1.0.3
- Screenshot was auto-assigned to custom category `Invoices`
- I changed the main category from `Documents` to `Receipts`
- Expected: custom category stays as `Invoices`
- Actual: custom category resets to `None`
- Happens every time on new captures
